## About

I build practical tools that turn scattered AI workflows into focused, usable products.

My recent work is centered on:

- AI image workbenches with OpenAI-compatible routes
- browser-first tooling that keeps the workflow in one place
- self-hosted utilities that are easier to deploy, test, and maintain
- clear boundaries between public UI, private infrastructure, and production data

## Featured Projects

### [AI Image Workbench](https://github.com/margetrp-hub/image-sub2api-studio)

A browser-first image creation workstation for OpenAI-compatible APIs and gateways.

It brings prompts, reference images, generation settings, canvas iteration, queues, and history into one focused workspace instead of spreading the workflow across multiple tools.

### [GPT Account Manager](https://github.com/margetrp-hub/gpt-account-manager)

A self-hosted operations workspace for mailbox intake, maintenance flows, queue handling, and account support tooling.

This project is more operations-oriented than the image workbench, so I treat deployment boundaries, data handling, and environment separation as first-class concerns.
